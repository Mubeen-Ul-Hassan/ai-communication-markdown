# Emphasis (Italic)

Emphasis, aka italics, with `*asterisks*` or `_underscores_`.

> *This is text is italics.*

# Strong (Bold)

Strong emphasis, aka bold, with `**asterisks**` or `__underscores__`.

> **This is text is bold.**

# Combined

Combined emphasis with `***asterisks***` and `*__underscores__*`.

> ***This is text is both bold and italics.***

# Strikethrough

Strikethrough uses two tildes. ``~~Strikethrough~~``.

> ~~This is text is strikethrough.~~

# Code

> To denote a word or phrase as code, enclose it in backticks (`).

Markdown | HTML | Rendered Output
:-- | :-- | :--
Type \`cd\` to change directory. | Type \<code>cd\</code> to change directory. | Type `cd` to change directory.

# Blockquote

Blockquotes can contain multiple paragraphs. Add a > on the blank lines between the paragraphs. Blockquotes can be nested. Add a >> in front of the paragraph you want to nest.

## Markdown

\> I think, therefore I am. <br>
\> <br>
\>> Know thyself.

## Rendered Output

> I think, therfore I am.
>
>> Know thyself.

# Paragraph

To create paragraphs, use a blank line to separate one or more lines of text.

Markdown | HTML | Rendered Output
:-- | :-- | :--
I really like using Markdown. | `<p>I really like using Markdown.</p>` | I really like using Markdown.
I will use it for all my documents. | `<p> I will use it for all my documents. </p>` | I will use it for all my documents.
